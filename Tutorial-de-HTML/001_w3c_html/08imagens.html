Imagens HTML

Imagens podem melhorar o design e a aparencia de uma página web.

Exemplo 1


                <img src="pic_trulli.jpg" alt="Italian Trulli">

Exemplo 2


                <img src="img_girl.jpg" alt="Girl in a jacket">

Exemplo 3


                <img src="img_chania.jpg" alt="Flowers in Chania">

Sintaxe de Imagens HTML

A tag HTML <img> é usada para incorporar uma imagem em uma página web.

Imagens não são tecnicamente inseridas em uma página web; imagens são vinculadas a páginas web. A etiqueta <img> cria um espaço para a imagem referenciada.

A tag <img> está vazia, contém apenas atributos e não tem uma tag de encerramento.

A etiqueta <img> exige dois atributos:

  • src - Especifica o caminho ate a imagem
  • alt - Especifica um texto alternativo para a imagem

Sintaxe


                <img src="URL" alt="Texto alternativo">

o atributo SRC

O atributo exigido src especifica o caminho (URL) para a imagem.

NOTA: Quando uma página web carrega, é o navegador que pega a imagem de um servidor web e a insere na página. Portanto, certifique-se que a imagem realmente permaneça no mesmo lugar em relação para a página da web, caso contrário, seus visitantes receberão um icone de link quebrado. O icone de link quebrado e o texto alt são mostrados caso o navegador não encontre a imagem.

Exemplo


                <img src="img_chania.jpg" alt="Flowers in Chania">

O atributo alternativo (alt)

O atributo alt requerido fornece um texto alternativo para uma imagem, se o usuario por algum motivo não consiga visualizá-lo (por causa da conexão lenta, um erro no SRC ou se o usuario usa um leitor de tela).

O valor do atributo alt deve descrever a imagem

Exemplo


                <img src="img_chania.jpg" alt="Flowers in chania">

Se um navegador não conseguir encontrar uma imagem, ele exibirá o valor do atributo alt.

Exemplo


                <img src="wrongname.gif" alt="Flowers in Chania">

DICA: Um leitor de tela é um programa de software que lê o codigo HTML e permite ao usuario "escutar" o conteudo. Leitores de tela são uteis para pessoas com deficiencia visual ou com dificuldades de aprendizagem.

Tamanho da imagem - Largura e Altura

Voce pode usar o atributo style para especificar a largura e altura de uma imagem.

Exemplo


                <img src="img_girl.jpg" alt="Girl in a Jacket" style="width: 500px; height: 600px;">

Outra alternativa, é usar os atributos width e height

Exemplo


            <img src="img_girl.jpg" alt="Girl in a jacket" width="500px" height="600px">

Os atributos width e height sempre definem a largura e altura da imagem em pixels.
