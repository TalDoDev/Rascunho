Cores HTML

A cores HTML são especificadas com nomes de cores pre definidas, ou com valores RGB, HEX, HSL, RGBA ou HSLA

Nomes das cores

Em HTML, uma cor pode ser especificada usando um nome de cor

Tomato

DodgerBlue

Gray

Violet

Orange

MediumSeaGreen

SlateBlue

Lightgray

Cor de fundo

Você pode definir a cor de fundo dos elementos HTML

Exemplo


            <h1 style="background-color:DodgerBlue;">Olá Mundo</h1>
            <p style="background-color:Tomato;">Lorem...</p>

Cor do texto

Voce pode definir a cor do texto:

Exemplo


            <h1 style="color:Tomato;">Olá Mundo</h1>
            <p style="color:DodgerBlue;">Lorem ipsum...</p>
            <p style="color:MediumSeaGreen;">Lorem ipsum...</p>

Cor da borda

Voce pode definir a cor das bordas:

Exemplo


            <h1 style="border:2px solid tomato;">Hello world</h1>
            <h1 style="border:2px solid Dodgerblue;">Hello World</h1>
            <h1 style="border:2px solid violet;">Hello World</h1>

Valores de cor

Em HTML, as cores também podem ser especificadas usando valores RGB, valores HEX, valores HSL, valores RGBA e valores HSLA.

Os três elementos <div> seguintes têm cor de fundo definida com valores RGB, HEX e HSL:

RGB(255, 99, 71)

#ff6347

HSL(9, 100%, 64%)

Os dois elementos <div> seguintes têm a cor de fundo definida com valores RGBA e valores HSLA, que adicionam um canal Alpha a cor (aqui temos 50% de transparência)

RGBA (255,99, 71, 0.5)

HSLA (9, 100%, 64%, 0.5)

Exemplo:


            <h1 style="background-color: rgb(255, 99, 71);">...</h1>
            <h1 style="background-color: #ff6347">...</h1>
            <h1 style="background-color: hsl(9, 100%, 64%)">...</h1>

            <h1 style="background-color: rgba(255, 99, 71, 0.5)">...</h1>
            <h1 style="background-color: hsla(9, 100%, 64%, 0.5)">...</h1>

Valores de cores RGB

Em HTML, uma cor pode ser especificada com um valor RGB, usando esta formula:
RGB (red, green, blue)

Cada parâmetro (vermelho, verde, azul) define a intensidade da cor com valor entre 0 e 255.

Por exemplo, rgb(255, 0, 0) é exibido como vermelho, porque o vermelho está definido para seu valor mais alto (255), e os outros dois (verde e azul) são definido para 0.

Outro exemplo, rgb (0, 255, 0) é exibido em verde, porque verde está definido para seu valor mais alto (255), e os outros dois (vermelho e azul) são definidos para 0.

Para exibir preto, defina todos os parâmetros de cor para 0, assim:
rgb(0, 0, 0)

Para exibir branco, defina todos os parâmetros de cor para 255, como:
rgb(255, 255, 255)

Exemplo:

RGB(255, 0, 0)

RGB(60, 179, 113)

RGB(255, 165, 0)

RGB(0, 0, 255)

RGB(238, 130, 238)

RGB(106, 90, 205)

Tons de cinza

Tons de cinza são frequentemente definidos usando valores iguais para os três parâmetros:

Exemplo:

RGB(60, 60, 60)

RGB(140, 140, 140)

RGB(200, 200, 200)

RGB(100, 100, 100)

RGB(180, 180, 180)

RGB(240, 240, 240)

Valores de cor RGBA

Valores de cor RGBA são uma extensão dos valores de cor RGB com um canal Alpha - que especifica a opacidade de uma cor.

Um valor de cor RGBA é especificado com:
RGBA(red, green, blue, alpha)

O parâmetro alfa é um número entre 0.0(totalmente transparente) e 1.0(nada transparente)

Exemplo:

rgba(255, 99, 71, 0)

rgba(255, 99, 71, 0.4)

rgba(255, 99, 71, 0.8)

rgba(255, 99, 71, 0.2)

rgba(255, 99, 71, 0.6)

rgba(255, 99, 71, 1)

Valores de cor HEX

Uma cor hexadecimal é especificada com: #RRGGBB, onde o RR (red), GG (green) e BB (blue) inteiros hexadecimais especificam os componetes da cor. São valores hexadecimais entre 00 e ff (igual aos decimais de 0 a 255).

Por exemplo, #ff0000 é exibido como vermelho, porque o vermelho é definido para seu valor mais alto (ff), e os outros dois (verde e azul) estão definidos para 00.

Outro exemplo, #00ff00 é exibido como verde, porque verde é definido para seu valor mais alto (ff), e os outros dois (vermelho e azul) são ajustados para 00.

Para exibir preto, defina todos os parâmetros de for para 00, como isto: #000000.

Para exibir branco, defina todos os parâmetros de cor para ff, como isto: #ffffff

Exemplo

#ff0000

#3cb371

#ffa500

#0000ff

#ee82ee

#6a5acd

Tons de cinza

Tons de cinza são frequentemente definidos usando valores iguais para os três parametros.

Exemplo:

#404040

#a0a0a0

#dcdcdc

#686868

#bebebe

#f8f8f8

Cores HSL e HSLA

HSL significa Hue, Saturation and Lightness (matiz, saturação e clareza)

Os valores de cor HSLA são uma extensão do HSL com um canal alfa (opacidade)

Em HTML, uma cor pode ser especificada usando matiz, saturação e clareza (HSL):
HSL (matiz, saturação, clareza)

Hue é um grau na roda colorida de 0 a 360, 0 é vemelho, 120 é verde e 240 é azul.

Saturação é um valor percentual, 0% significa umtom de cinza, e 100% é a cor completa.

A leveza também é um valor percentual, 0% é preto e 100% é branco.

Exemplo:

HSL(0, 100%, 50%)

HSL(147, 50%, 47%)

HSL(39, 100%, 50%)

HSL(240, 100%, 50%)

HSL(300, 76%, 72%)

HSL(248, 53%, 58%)

Saturação

Saturação pode ser descrita como a intensidade de uma cor.

100% é cor pura, sem tons de cinza.

50% é 50% cinza, mas ainda dá para ver a cor.

0% é completamente cinza, você não consegue mais ver a cor.

Exemplo:

HSL(0, 100%, 50%)

HSL(0, 60%, 50%)

HSL(0, 20%, 50%)

HSL(0, 80%, 50%)

HSL(0, 40%, 50%)

HSL(0, 0%, 50%)

Leveza

A claridade de uma cor pode ser descrita como a quantidade de luz que voce quer dar à cor, onde 0% significa sem luz (preto), 50% significa 50% claro (nem escuro nem claro), e 100% significa clareza total (branco).

Exemplo:

HSL(0, 100%, 0%)

HSL(0, 100%, 50%)

HSL(0, 100%, 90%)

HSL(0, 100%, 25%)

HSL(0, 100%, 75%)

HSL(0, 100%, 100%)

Tons de cinza

Tons de cinza são frequentemente definidos ajustando o matiz e a saturação em 0, e ajustando a clareza de 0% para 100% para obter tons mais escuros/claros:

Exemplo:

HSL(0, 0%, 20%)

HSL(0, 0%, 40%)

HSL(0, 0%, 70&)

HSL(0, 0%, 30%)

HSL(0, 0%, 60%)

HSL(0, 0%, 90%)

Valores de cor HSLA

Os valores de cor HSLA são uma extensão dos valores de cor HSL, com um canal alfa (opacidade) de uma cor.

Um valor de cor HSLA é especificado com:
HSLA(matiz, saturação, clareza, alfa)

O parâmetro alfa é um numero entre 0.0 (totalmente transparente) e 1.0 (nada transparente):

Exemplo:

HSLA(9, 100%, 64%, 0)

HSLA(9, 100%, 64%, 0.4)

HSLA(9, 100%, 64%, 0.8)

HSLA(9, 100%, 64%, 0.2)

HSLA(9, 100%, 64%, 0.6)

HSLA(9, 100%, 64%, 1)
