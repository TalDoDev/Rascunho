Links HTML

Links estão presentes em quase todas as páginas da web. Links permitem que os usuarios cliquem de página em página.

Links HTML - Hiperlinks

Links HTML são hiperlinks.

Você pode clicar em um link e pular para outro documento.

NOTA: Um link não precisa ser texto. Um link pode ser uma imagem ou qualquer outro elemento HTML!

Links HTML - Sintaxe

A tag HTML <a> define um hiperlink. Possui a seguinte sintaxe:


                <a href="url">Conteudo do link</a>

O atributo mais importante do elemento <a> é o atributo href, que indica o destino do link.

O texto do link é a parte que ficava visivel para o leito.

Clicar no texto do link, o leitor será redirecionado para o endereço URL especificado.

Exemplo

Este exemplo mostra como criar um link para W3Schools.com!


                <a href="https://www.w3schools.com/">Visite W3Schools!</a>

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

  • Um link não visitado está sublinhado e azul
  • Um link visitado está sublinhado e roxo
  • Um link ativo é sublinhado e vermelho

Links HTML - O atributo Alvo(target)

Por padrão, a página vinculada será exibida na janela atuaç do navegador. Para mudar isso. voce deve especificar outro alvo para o link.

O atributo target especifica onde abrir o documento vinculado

O atributo target pode ter um dos seguintes valores:

  • _self - Padrão. Abre o documento na mesma janela/aba em que foi clicada
  • _blank - Abre o documento em uma nova janela ou aba
  • _parent - Abre o documento no frame pai
  • _top Abre o documento no corpo inteiro da janela

Exemplo

Use target="_blank" para abrir o documento vinculado em uma nova janela ou aba do navegador:


                <a href="https://www.w3schools.com/" target="_blank">Visite W3Schools!</a>

URLs absolutas vs. URLs relativas

Ambos os exemplo acima usam uma URL absoluta (um endereço web completo) no atributo href

Um link local (um link para uma pagina dentro do mesmo site) é especificado com uma URL relativa (sem a parte do "https://www..."):

Exemplo


                <h2>URL Absoluta</h2>
                <p><a href="https://www.w3.org/">W3C</a></p>
                <p><a href="https://www.google.com">Google</a></p>

                <h2>URL Relativa</h2>
                <p><a href="html_images.asp">HTML Images</a></p>
                <p><a href="/css/default.aasp">CSS Tutorial</a></p>

Links HTML - Use imagens como link

Para usar uma imagem como link, basta colocar a tag <img> dentro da tag <a>

Exemplo


                <a href="default.asp">
                <img src="smiley.gif" alt="HTML tutorial" style="width: 42px; height: 42px;">
                </a>

Links para um endereço de E-mail

Use dentro do atributo href para criar um link que abra o programa de e-mail do usuario mailto (para que eles enviem um novo email):

Exemplo


                <a href="mailto: someone@example.com">Send email</a>

Botão como Link

Para usar um botão HTML como link, voce precisa adicionar um pouco de código JavaScript.

O JavaScript permite que voce especifique o que acontece em certos eventos, como um clique em um botão:

Exemplo


                <button onclick="document.location='default.asp'">HTML Tutorial</button>

Titulos de Link

O atributo title especifica informações extras sobre um elemento. A informação é mais frequentemente mostrada como um texto de dica quando o mouse passa sobre o elemento.

Exemplo


                <a href="https://www.w3schools.com/html/" title="Go to W3Schools HTML section">Visit our HTML Tutorial</a>

Mais sobre URLs Absolutas e URLs Relativas

Exemplo 1

Use uma URL completa para linkar para uma página web:

<a href="https://www.w3schools.com/html/default.asp">HTML tutorial</a>

Exemplo 2

Link para uma página localizada na pasta html do site atual:


                <a href="/html/default.asp">HTML tutorial</a>

Exemplo 3

Link para uma página localizada na mesma pasta da página atual:


                <a href="default.asp">HTML Tutorial</a>

Resumo do Capítulo

  • Use o elemento <a> para definir um link
  • Use o atributo href para definir o endereço do link
  • Use o atributo target para definir onde abrir o documento vinculado
  • Use o elemento <img> (dentro de <a>) para usar uma imagem como link
  • Use o esquema mailto: dentro do atributo href para criar um link que abra o programa de e-mail do usuario

Links HTMK - Cores diferentes

Um link HTML é exibido em uma cor diferente dependendo se já foi visitado, não visitado ou ativo

Cores do link HTML

Por padrão, um link aparecerá assim (em todos os navegadores):

  • Um link não visitado está sublinhado e azul
  • Um link visitado está sublinhado e roxo
  • Um link ativo é sublinhado e vermelho

Voce pode mudar as cores do estado do link usando CSS:

Exemplo

Aqui, um link não visitado será verde, sem sublinhado. Um link visitado será rosa, sem sublinhado. Um link ativo será amarelo e sublinhado. Alem disso, ao passar o mouse sobre um link (a:hover), ele fica vermelho e sublinhado:


                <style>
                    a:link {
                        color: green;
                        background-color: transparent;
                        text-decoration: none;
                    }
                    a:visited {
                        color: pink;
                        background-color: transparent;
                        text-decoration: none;
                    }
                    a:hover {
                        color: red;
                        background-color: transparent;
                        text-decoration: underline;
                    }
                    a:active {
                        color: yellow;
                        background-color: transparent;
                        text-decoration: underline;
                    }
                </style>

Links HTML - Criar favoritos

Links HTML podem ser usados para criar favoritos, para que os leitores possam ir para partes especificas de uma pagina web.

Criar um favorito em HTML

Os marcadores podem ser úteis se uma página da web for muito longa.

Para criar um marcador - primeiro crie o favorito e depois adicione um link para isso.

Quando o link é clicado, a página desce ou sobe até o local com o Favorito.

Exemplo

Primeiro, use o atributo id para criar um favorito


                <h2 id="C4">Chapter 4</h2>

Depois, adicione um link para o favorito (Um link para pular pro chapter 4), dentro da mesma pagina:

Exemplo


                <a href="#C4">Subir até o capitulo 4</a>

Voce tambem pode adicionar um link para um favorito em outra página:

Exemplo


                <a href="html_demo.html#C4">Pule para o capitulo 4</a>

Volte para o inicio da pagina

Resumo do capitulo

  • Use o atributo id (id="valor") para definir favoritos em uma página
  • Use o atributo href (href="#valor") para linkar ao favorito
