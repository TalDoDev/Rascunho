Estilos HTML - CSS

Folhas de Estilo em Cascata

CSS significa Cascading Style Sheets(folhas de Estilo em Cascata).

CSS econimiza muito trabalho. Ele pode controlar o layout de múltiplas páginas da web todas ao mesmo tempo.

O que é CSS?

Folhas de Estilo em Cascata (CSS) são usadas para formatar o layout de uma página web.

Com CSS, você pode controlar a cor, a fonte, o tamanho do texto, o espaçamento entre os elementos, como os elementos são posicionados e dispostos, qual contexto imagens ou cores de fundo devem ser usadas, com diferentes displays para dispositivos diferentes e tamanhos de tela, e muito mais.

Usando CSS

O CSS pode ser adicionado a documentos HTML de 3 maneiras:

  1. Inline - Usando o atributo style dentro dos elementos HTML
  2. Interno - Usando um elemento <style> na seção <head>
  3. Externo - Usando um elemento <link> para vincular a um arquivo CSS

A forma mais comum de adicionar CSS é manter os estilos em CSS em arquivos externos.

No entanto, neste tutorial vamos usar estilos inline e interno, porque isso é mais fácil de demonstrar, e é mais facil para voce tentar sozinho.

Inline CSS

Um CSS inline é usado para aplicar um estilo único a um único elemento HTML.

Um CSS inline usa o atributo style dentro de um elemento HTML.

O exemplo a seguir define a cor do texto do elemento <h1> para azul, e a cor do texto do elemento <p> para vermelho:

Exemplo:


                <h1 style="color: blue;">Um titulo em azul</h1>
                <p style="color: red;">Um parágrafo em vermelho</p>

CSS interno

Um CSS interno é usado para definir um estilo para uma única página HTML.

Um CSS interno <style> é definido na seção de uma página HTML, dentro de um elemento <head>

Exemplo


                <!DOCTYPE html>
                <html>
                <head>
                <style>
                    body {
                        background-color: powderblue;
                        color: beige;
                    }
                    h1 {
                        font-family: Arial;
                        color: blue;
                    }
                    p {
                        color: red;
                    }
                    ...
                </style>
                </head>
                <body>
                    <h1>Titulo...</h1>
                    <p>Lorem...</p>
                </body>
                </html>

CSS Externo

Uma folha de estilo externa é usada para definir o estilo de muitas páginas HTML.

Para usar uma folha de estilo externa, adicione um link para ela na seção <head> de cada página HTML:

Exemplo


                <!DOCTYPE html>
                <html>
                <head>
                <link rel="stylesheet" href="styles.css">
                </head>
                <body>
                    <h1>Titulo...</h1>
                    <p>Lorem...</p>
                </body>
                </html>

A folha de estilo externa pode ser escrita em qualquer editor de texto, O arquivo não deve conter nenhum código HTML, e deve ser salvo com uma extensão .css.

Aqui está como é o arquivo "styles.css"

Styles.css


                @charset "UTF-8";
                body {
                    background-color: powderblue;
                }
                h1 {
                    color: blue;
                }
                p {
                    color: red;
                }

Cores, Fontes e Tamanhos CSS

Aqui, vamos demonstrar algumas propriedades CSS muito usadas.

A propriedade CSS color define a cor do texto a ser utilizada.

A propriedade CSS font-family define a fonte a se usada.

A propriedade CSS font-size define o tamanho do texto a ser utilizada.

Exemplo

Uso das propriedades CSS color, font-family e font-size.


                <!DOCTYPE html>
                <head>
                <style>
                    h1 {
                        color: blue;
                        font-family: verdana;
                        font-size: 300%;
                    }
                    p {
                        color: red;
                        font-family: courier;
                        font-size: 160%;
                    }
                </style>
                </head>
                <body>
                <h1>Titulo</h1>
                <p>Lorem...</p>
                </body>
                </html>

CSS Border

A propriedade CSS border define uma borda em torno de um elemento HTML.

Exemplo

Uso da propriedade CSS border.


                p {
                    border: 2px solid black;
                }

Preenchimento CSS

A propriedade CSS padding define um preenchimento (espaço) entre o texto e a borda.

Exemplo

Uso da propriedade CSS border.


                p {
                    border: 1px solid black;
                    padding: 30px;
                }

Margem CSS

A propriedade CSS margin define uma margem (espaço) fora da bordar

Exemplo

Uso da propriedade CSS border.


                p {
                    border: 1px solid black;
                    margin: 50px;
                }

Folhas para CSS externo

Folhas de estilo externas podem ser referenciadas com uma URL completa ou com um camilho relativo a pagina web atual.

Exemplo 1:

Este exemplo usa uma URL completa para linkar a uma folha de estilos.


                <link rel="stylesheet" href="https://www.w3schools.com/html/styles.css">

Exemplo 2:

Este exemplo faz link para uma folha de estilo localizada na pasta html do site atual.


                <link rel="stylesheet" href="/html/styles.css">

Exemplo 3:

Este exemplo faz link para uma folha de estilo localizada na mesma pasta da página atual.


                <link rel="stylesheet" href="styles.css">

Resumo do capítulo

  • Use o atributo HTML style para estilo inline
  • Use o elemento HTML <style> para definir CSS interno
  • Use o elemento HTML <link> para se referir a um arquivo CSS externo
  • Use o elemento HTML <head> para armazenar elementos <style> e <link>
  • Use a propriedade CSS color para cores de texto
  • Use a propriedade CSS font-family para fontes de texto
  • Use a propriedade CSS font-size para tamanhos de texto
  • Use a propriedade CSS border para bordas
  • Use a propriedade CSS padding para o espaço dentro da borda
  • Use a propriedade CSS margin para espaço fora da borda
