Estilos HTML

O atributo HTML style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e mais.

O atributo HTML style

Definir o estilo de um elemento HTML pode ser feito com o atributo: style.

O atributo HTML style possui a seguinte sintaxe:

Exemplo:


            <tagname style="propriedade:valor;">

A propriedade é uma propriedade CSS. O valor é um valor CSS

Cor de fundo

A propriedade CSS background-color define a cor de fundo para um elemento HTML.

Exemplo 1:

Defina a cor de fundo de uma página para azul-claro:


            <body style="background-color:powderblue;">
                <h1>Este é um titulo</h1>
                <p>Isto é um paragrafo</p>
            </body>

Exemplo 2:

Defina a cor de fundo para dois elementos diferentes:


            <body>
                <h1 style="background-color:powderblue;">Titulo com a cor de fundo azul-claro</h1>
                <p style="background-color:tomato;">Isto é um paragrafo com a cor de fundo vermelha.</p>

Cor de Texto

A propriedade CSS color define a cor do texto para um elemento HTML:

Exemplo


            <h1 style="color:blue;">Titulo com a cor da fonte azul</h1>
            <p style="color:red;">Isto é um paragrafo com a cor da fonte vermelha.</p>

Fontes

A propriedade CSS font-family define a fonte a ser usada para um elemento HTML

Exemplo:


            <h1 style="font-family:verdana;">Titulo com a fonte alterada</h1>
            <p style="font-family:courier;">Isto é um parágrafo com a fonte alterada.</p>

Tamanho de Texto

A propriedade CSS font-size define o tamanho do texto para um elemento HTML

Exemplo:


            <h1 style="font-size:300%;">Titulo com o tamanho da fonte modificado</h1>
            <p style="font-size:160%;">Isto é um parágrafo com o tamanho da fonte modificado.</p>

Alinhamento do Texto

A propriedade CSS text-align define o alinhamento horizontal do texto para um elemento HTML:

Exemplo:


            <h1 style="text-align:center;">Titulo centralizado</h1>
            <p style="text-align:center;">Paragrafo centralizado</p>

Resumo do Capítulo

  • Use o atributo style para estilizar elementos HTML.
  • Use background-color para cor de fundo
  • Use color para cores de texto
  • Use font-family para fontes de texto
  • Use font-size para tamanhos de texto
  • Use text-align para alinhamento de texto
