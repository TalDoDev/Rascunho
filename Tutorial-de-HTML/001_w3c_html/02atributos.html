Atributos HTML

  • Todos os elementos HTML podem ter atributos
  • Atributos fornecem informações adicionais sobre elementos
  • Os atributos são sempre especificados na tag inicial
  • Atributos geralmente vêm em pares nome/valor como: name="value"

O atributo href

A tag <a> define um hiperlink. O atributo href especifica a URL da página que o link vai

Exemplo:


            <a href="https://www.w3schools.com">Visite W3Schools</a>

O atributo src

A tag <img> é usada para incorporar uma imagem em uma página HTML. O atributo src especifica o caminho até a imagem a ser exibida

Exemplo:


            <img src="img_girl.jpg">

Existem duas maneiras de especificar a URL do atributo src

  1. URL absoluta - Links para uma imagem externa hospedada em outro site.
    Exemplo: src="https://www.w3schools.com/images/img_girl.jpg"
  2. URL relativa - Links para uma imagem interna, hospedada dentro de sua maquina. Aqui a URL não inclui o nome de domínio. Se a URL começar sem barra, será relativa a página atual.
    Exemplo: src="img_girl.jpg".
    Se a URL começar com barra, ela será relativa ao dominio.
    Exemplo: src="/images/img_girl.jpg.

NOTAS: Imagens externas podem estar protegidas por direitos autorais. Se você fizer e não conseguir permissão para usá-la, voce pode estar violando as leis de direitos autorais. Além disso, você não pode controlar imagens externas; pode ser removido de repente ou mudado.

Os atributos width e height

A tag <img> também pode conter os atributos width e height

Exemplo:


            <img src="img_girl.jpg" width="500px" height="600px">

O atributo alternativo (alt)

O atributo alt exigido para a tag <img> especifica um texto alternativo para uma imagem, caso a imagem por algum motivo não possa ser exibida. Isso pode ser devido a uma conexão lenta, ou um erro no atributo src, ou se o usuário usa tela de leitor.

Exemplo:


            <img src="img_girl.jpg" alt="Girl with a jacket">

O atributo style

O atributo style é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e mais.

Você vai aprender mais sobre estilos em nosso capitulo de Estilos HTML.

O atributo lang

Você sempre deve incluir o atributo lang dentro do elemento <html>, para declarar o idioma da página da web. Isso tem como objetivo auxiliar mecanismos de busca e navegadores.

O exemplo a seguir especifica o portugues como idioma:

Exemplo:


            <!DOCTYPE html>
            <html lang="pt-br">
            <body>
            ...
            </body>
            </html>

O atributo title

O atributo title define algumas informações extras sobre um elemento.

O valor do atributo title será exibido como uma dica de ferramenta quando passar com o mouse sobre o elemento.

Exemplo:


            <p title="I'm a tooltip">Isto é um paragrafo.</p>
