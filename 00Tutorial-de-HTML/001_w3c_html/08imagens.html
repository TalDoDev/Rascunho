Imagens HTML

Imagens podem melhorar o design e a aparencia de uma página web.

Exemplo 1


                <img src="pic_trulli.jpg" alt="Italian Trulli">

Exemplo 2


                <img src="img_girl.jpg" alt="Girl in a jacket">

Exemplo 3


                <img src="img_chania.jpg" alt="Flowers in Chania">

Sintaxe de Imagens HTML

A tag HTML <img> é usada para incorporar uma imagem em uma página web.

Imagens não são tecnicamente inseridas em uma página web; imagens são vinculadas a páginas web. A etiqueta <img> cria um espaço para a imagem referenciada.

A tag <img> está vazia, contém apenas atributos e não tem uma tag de encerramento.

A etiqueta <img> exige dois atributos:

  • src - Especifica o caminho ate a imagem
  • alt - Especifica um texto alternativo para a imagem

Sintaxe


                <img src="URL" alt="Texto alternativo">

o atributo SRC

O atributo exigido src especifica o caminho (URL) para a imagem.

NOTA: Quando uma página web carrega, é o navegador que pega a imagem de um servidor web e a insere na página. Portanto, certifique-se que a imagem realmente permaneça no mesmo lugar em relação para a página da web, caso contrário, seus visitantes receberão um icone de link quebrado. O icone de link quebrado e o texto alt são mostrados caso o navegador não encontre a imagem.

Exemplo


                <img src="img_chania.jpg" alt="Flowers in Chania">

O atributo alternativo (alt)

O atributo alt requerido fornece um texto alternativo para uma imagem, se o usuario por algum motivo não consiga visualizá-lo (por causa da conexão lenta, um erro no SRC ou se o usuario usa um leitor de tela).

O valor do atributo alt deve descrever a imagem

Exemplo


                <img src="img_chania.jpg" alt="Flowers in chania">

Se um navegador não conseguir encontrar uma imagem, ele exibirá o valor do atributo alt.

Exemplo


                <img src="wrongname.gif" alt="Flowers in Chania">

DICA: Um leitor de tela é um programa de software que lê o codigo HTML e permite ao usuario "escutar" o conteudo. Leitores de tela são uteis para pessoas com deficiencia visual ou com dificuldades de aprendizagem.

Tamanho da imagem - Largura e Altura

Voce pode usar o atributo style para especificar a largura e altura de uma imagem.

Exemplo


                <img src="img_girl.jpg" alt="Girl in a Jacket" style="width: 500px; height: 600px;">

Outra alternativa, é usar os atributos width e height

Exemplo


            <img src="img_girl.jpg" alt="Girl in a jacket" width="500px" height="600px">

Os atributos width e height sempre definem a largura e altura da imagem em pixels.

Largura e altura, ou estilo?

Os atributos width, height e style são todas válidas em HTML.

No entanto, sugerimos usar o atributo style. Ele impede que as folhas de estilos mudem o tamanho das imagens

Exemplo


                <!DOCTYPE html>
                <html>
                <head>
                <style>
                    img {
                        width: 100%;
                    }
                </style>
                </head>
                <body>
                    <img src="html5.gif" alt="HTML5 Icon" width="128" height="128">
                    <img src="html5.gif" alt="HTML5 Icon" style="width: 128px; height: 128px;">
                </body>
                </html>

Imagens em outra pasta

Se voce tiver suas imagens em outra subpasta, deve incluir o nome da pasta no atributo src

Exemplo


                <img src="/imagens/html5.gif" alt="HTML5 Icon" style="width: 128px; height: 128px;">

Imagens em outro servidor/site

Alguns sites apontam para uma imagem em outro servidor.

Para apontar para uma imagem em outro servidor, voce deve especificar uma URL absoluta (URL completa) no atributo src

Exemplo


                <img src="https://www.w3schools.com.../..." alt="W3Schools.com">

Imagens Animadas

HTML permite GIFs animados

Exemplo


                <img src="programming.gif" alt="Computer Man" style="width: 48px; height: 48px;">

Imagem como link

Para usar uma imagem como link, coloque a tag <img> dentro da tag <a>

Exemplo


                <a href="default.asp">
                    <img src="smiley.gif" alt="HTML Tutorial" style="width: 42px; height: 42px;">
                </a>

Imagem Flutuante

Use aa propriedade CSS float para deixar a imagem flutuar para a direita ou para a esquerda de um texto.

Exemplo


                <p><img src="smiley.gif" alt="Smiley face" style="float: right; width: 42px; height: 42px;"> The image will float to the right of the text.</p>
                <p><img src="smiley.gif" alt="Smiley face" style="float: left; width: 42px; height: 42px;"> The image will float to the left of the text.</p>

Mapas de Imagens HTML

Com mapas de imagens em HTML, voce pode criar areas clicaveis em uma imagem

Mapas de imagens

A tag HTML <map> define um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicaveis. As areas são definidas com uma ou mais tags <area>

Exemplo

Aqui está o código-fonte HTML do mapa de imagem acima:


                <img src="workplace.jpg" alt="Workplace" usemap="#workmap">
                <map name="workmap">
                    <area shape="rect" coords="34,44,270,350" href="00home.html" alt="Computer">
                    <area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
                    <area shape="circle" coords="337,300,44" alt="Coffee" href="coffee.htm">
                </map>

Como funciona?

A ideia por tras de um mapa de imagem é que voce deve conseguir realizar diferentes performances dependendo de onde voce clicar na imagem.

Para criar um mapa de imagem, voce precisa de uma imagem e de algum código HTML que descreva as áreas clicáveis.

A imagem

A imagem é inserida usando tag <img>. A única diferença em relação a outras imagens é que voce deve adicionar um atributo usemap


            <img src="workplace.jpg" alt="workplace" usemap="#workmap">

O valor começa com uma hashtag seguida pelo nome do mapa da imagem, e é usado para criar uma relação entre a imagem e o mapa da imagem

Criar mapa de imagem

Depois, adicione um elemento <map>

O elemento <map> é usado para criar um mapa de imagem, e é vinculado à imagem usando o atributo obrigatório name


            <map name="workmap">

O atributo name deve ter o mesmo valor que o atributo usemap de <img>

As Áreas

Depois, adicione as áreas clicáveis.

Uma área clicavel é definida usando um elemento <area>

Formato

Voce deve definir o formato da area clicavel, e pode escolher um desses valores:

  • rect - define uma regiao retangular
  • circle - define uma regiao circular
  • poly - define uma regiao poligonal
  • default - define toda a regiao

Voce tambem deve definir algumas coordenadas para poder posicionar a area clicavel a imagem.

Shape="rect"

As coordenadas para shape="rect" vem em pares, uma para o eixo X e outra para o eixo Y.

Então, as coordenadas estão localizadas a 34 pixels da margem esquerda e 44 pixel do topo.

As outas coordenadas estão localizadas em 270 pixels da margem ESQUERDA e 350 pixels do TOPO.

Agopra temos dados suficientes para criar uma areas retangulas cliclavel

Exemplo

            <area shape="rect" coords="34, 44, 270, 350" href="computer.html">

Shape="circle"

Para adicionar uma area circular, primeiro localize as coordenadas do centro do circulo: 337, 300.

Então especifique o raio do circulo: 44 pixel.

Agora voce tem dados suficiente para criar uma area circular clicavel

Exemplo

                <area shape="circle" coords="337, 300, 44" href="coffee.html">

Shape="poly"

O shape="poly" contém vários pontos de coordenadas, que criam uma forma formada por linhas retas (um poligono).

Isso pode ser usado para criar qualquer forma.

Tipo, um croissant!

Como podemos fazer o croissant na imagem abaixo virar um link clicável?

Temos que encontrar as coordenadas X e Y para todas as arestas do croissant:

As coordenadas vem em pares, uma para o eixo X e a outra para o eixo Y:

Exemplo

                <area shape="poly" coords="140, 121, 181, 116, 204, 160, 204, 222, 191, 270, 140, 329, 85, 355, 58, 352, 37, 322, 40, 259, 103, 161, 128, 147" href="croissant.html">

Mapa de Imagem e JavaScript

Uma área clicavel tambem pode disparar uma função JavaScript.

Adicionar um evento click ao elemento <area> para executar uma função JavaScript

Exemplo

Aqui, usamos o atributo onclick para executar uma função JavaScript quando selecionar a area:


                <map name="workmap">
                    <area shape="circle" coords="337, 300, 44" href="coffee.html" onclick="myFunction()">
                </map>

                <script>
                    function myFunction() {
                        alert("You clicked the coffee cup!");
                    }
                </script>

Resumo do capitulo

  • Use o elemento HTML <map> para definir um mapa de imagem
  • Use o elemento HTML <area> para definir as areas clicaveis no mapa da imagem
  • Use o atributo HTML usemap do elemento <img> para apontar para um mapa de imagem

Imagens de fundo HTML

Uma imagem de fundo pode ser especificada para quase todo elemento HTML

Imagem de fundo em um elemento HTML

Para adicionar uma imagem de fundo em um elemento HTML, use o atributo HTML style e a propriedade CSS background-image

Exemplo

Adicione uma imagem de fundo em um elemento <p>


                <p style="background-image: url('img_girl.jpg');">

Voce tambem pode especificar a imagem de fundo no elemento <style>, na seção <head>

Exemplo

Especifique a imagem de fundo no elemento <style>


                <style>
                    p {
                        background-image: url('img_girl.jpg');
                    }
                </style>

Imagem de fundo em uma pagina

Se voce quer que a pagina inteira tenha uma imagem de fundo, precisa especificar a imagem de fundo do elemento <body>
