Tutorial de HTML

Aprenda HTML

HTML é a linguagem padrão de marcação para páginas Web.

Com HTML, você pode criar seu próprio site.

Introdução ao HTML

O que é HTML?

  • HTML significa Linguagem de Marcação de HiperTexto
  • HTML é a linguagem padrão de marcação para criar páginas Web
  • HTML descreve a estrutura de uma página Web
  • HTML consiste em uma série de elementos
  • Elementos HTML indicam ao navegador como exibir o conteúdo
  • Elementos HTML rotulam pedaços de conteúdo como "este é um título", "isto é um parágrafo", "este é um link", etc...

Um documento HTML simples

Exemplo


                <!DOCTYPE html>
                <html>
                <head>
                <title>Titulo da pagina</title>
                </head>
                <body>
                    <h1>Este é um titulo</h1>
                    <p>Isto é um parágrafo</p>
                </body>
                </html>

Exemplo explicado

  • A declaração <!DOCTYPE html> define que este documento é um documento HTML5
  • O elemento <html> é o elemento raiz de uma página HTML
  • O elemento <head> contém metainformações sobre a página HTML
  • O elemento <title> especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na aba da página)
  • O elemento <body> é o contêiner para todos os conteúdos visíveis, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc...
  • O elemento <h1> define um título grande
  • O elemento <p> define um parágrafo

O que é um elemento HTML?

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final:

<TagInicial>O conteúdo vai aqui...</TagFinal>

Tag Inicial Conteúdo do Elemento Tag Final
<h1> Conteúdo do titulo </h1>
<p> Conteúdo do parágrafo </p>
<br> none none

NOTA: Alguns elementos HTML não têm conteúdo (como o <br>). Esses elementos são chamados de elementos vazios. Elementos vazios não têm tag final!

Elementos HTML aninhados

Elementos HTML podem ser aninhados (isso significa que elementos podem conter outros elementos).

Todos os documentos HTML consistem em elementos HTML aninhados.

O exemplo a seguir contém quatro elementos HTML (<html>, <body>, <h1> e <p>)

Exemplo


                <!DOCTYPE html>
                <html>
                <body>
                    <h1>Este é um titulo</h1>
                    <p>Isto é um parágrafo</p>
                </body>
                </html>

Exemplo eplicado

  • O elemento <html> é o elemento raiz e define todo o documento HTML
  • Ele tem uma tag de início <html> e uma tag de fim </html>
  • Então, dentro do elemento <html> há um elemento <body>
  • O elemento <body> define o corpo do documento
  • Ele tem uma tag de inicio <body> e uma tag de fim </body>
  • Então, dentro do elemento <body> ali são outros dois elementos <h1> e <p>
  • O elemento <h1> define um titulo
  • Ele possui uma tag de início <h1> e uma tag de fim </h1>
  • O elemento <p> define um parágrafo
  • Ele possui uma tag de início <p> e uma tag de fim </p>

Elementos HTML vazios

Elementos HTML sem conteúdo são chamados de elementos vazios.

A tag <br> define uma quebra de linha, e é um elemento vazio sem uma tag de fim

Exemplo


                <p>Isto é um paragrafo <br> com uma quebra de linha</p>
