Exemplos Básicos de HTML

Neste capítulo, mostraremos alguns exemplos básicos de HTML

Não se preocupe se usarmos etiquetas que você ainda não conhece.

Documentos HTML

Todos os documentos HTML devem começar com uma declaração do tipo de documento: <!DOCTYPE html>

O próprio documento HTML começa com <html> e termina com </html>

A parte visível do documento HTML está entre <body> e </body>

Exemplo:


            <!DOCTYPE html>
            <html>
            <body>
                <h1>Titulo principal</h1>
                <p>Conteúdo do parágrafo</p>
            </body>
            </html>

Títulos HTML

Os títulos HTML são definidos com as tags <h1> até <h6>

<h1> Define o título mais importante.
<h6> Define o título menos importante

Exemplo:


                <h1>Titulo 1<:/h1>
                <h2>Titulo 2<:/h2>
                <h3>Titulo 3<:/h3>
                <h4>Titulo 4<:/h4>
                <h5>Titulo 5<:/h5>
                <h6>Titulo 6<:/h6>

NOTA: Os navegadores automaticamente adicionam algum espaço em branco (uma margem) antes e depois de um titulo.

Os títulos são importantes

Os mecanismos de busca usam os títulos para indexar a estrutura e o conteúdo das suas páginas web.

Os usuarios frequentemente folheiam uma página pelos títulos. É importante usar títulos para mostrar a estrutura do documento.

Títulos <h1> devem ser usados para títulos principais, seguidos pelos titulos <h2>, depois pelos menos importantes <h3>, e assim por diante. Até chegar no titulo <h6>

Por exemplo:

  • <h1> - Título da página
  • <h2> - Titulo de seções
  • <h3> - Subseções
  • ...

Exemplo:


                <h1>Travel Guide</h1>

                <h2>Europe</h2>
                <h3>France</h3>
                <h3>Italy<h3>
                
                <h2>Asia</h2>
                <h3>India</h3>
                <h3>Thailand</h3>

DICA: Use apenas um <h1> por página - ele representa o tema ou titulo principal

NOTA: Use títulos HTML apenas para títulos. Não use titulos para deixar o texto grande ou em negrito.

Títulos maiores

Cada título HTML tem um tamanho padrão. No entanto, você pode especificar o tamanho de qualquer titulo com o atributo style usando a propriedade CSS font-size

Exemplo:


                <h1 style="font-size::60px;">Titulo 1</h1>

Parágrafos HTML

Parágrafos HTML são definidos com a tag <p>

Um parágrafo sempre começa em uma nova linha, e os navegadores automaticamente adicionam algum espaço em branco (uma margem) antes e depois de um paragrafo.

O problema do poema

Este poema será exibido em uma única linha:

Exemplo


                <p>
                    My Bonnie lies over the ocean.

                    My Bonnie lies over the sea.

                    My Bonnie lies over the ocean.

                    Oh, bring back my Bonnie to me.
                </p>

Solução - O elemento HTML <pre>

O elemento HTML <pre> define um texto pré-formatado.

O texto dentro de um elemento é exibido em uma fonte de largura fixa (geralmente Courier), e preserva tanto os espaços quanto as quebras de linha:

Exemplo:


                <pre>
                    My Bonnie lies over the ocean.

                    My Bonnie lies over the sea.

                    My Bonnie lies over the ocean.

                    Oh, bring back my Bonnie to me.
                </pre>

Tags Vazias

Regras Horizontais HTML

A tag <hr> define uma quebra temática em uma página HTML e é mais frequentemente exibido como uma regra horizontal.

A tag <hr> é uma tag vazia, o que significa que não tem tag final.

Quebras de Linha HTML

O elemento HTML <br> define uma quebra de linha.

Use <br> se quiser uma quebra de linha (uma linha nova) sem começar um novo parágrafo.

A tag <br> é uma tag vazia, o que significa que não tem tag final.

Formatação de Texto HTML

O HTML contém vários elementos para definir texto com um significado especial.

Elementos de Formatação HTML

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

  • <b> - Texto em negrito
  • <strong> - Texto importante
  • <i> - Texto itálico
  • <em> - Texto enfatizado (enfase)
  • <mark> - Texto marcado
  • <small> - Texto menor
  • <del> - Texto excluido
  • <ins> - Texto inserido
  • <sub> - Texto subíndice
  • <sup> - Texto sobrescrito

Elementos HTML <b> e <strong>

O elemento HTML <b> define um texto em negrito, sem importância extra.

O elemento HTML <strong> define o texto com grande importância. O conteúdo interno geralmente é exibido em negrito.

Elementos HTML <i> e <em>

O elemento HTML <i> é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome, etc.... O conteúdo interno normalmente é exibido em Itálico

O elemento HTML <em> define enfase no texto. O conteudo interno normalmente tambem é exibido em itálico.

DICA: Um leitor de tela pronuncia palavras com ênfase, usando acensão verbal.

Elemento HTML <small>

O elemento HTML <small> define um texto menor

Exemplo


                <small>Este é um texto menor!</small>

Elemento HTML <mark>

O elemento HTML <mark> define o texto que devem ser marcados ou destacados

Exemplo:


                <p>Não se esqueça de comprar <mark>leite</mark> hoje</p>

Elemento HTML <del>

O elemento HTML <del> define o texto isso foi excluido de um documento. Os navegadores geralmente encontram uma linha através do texto deletado.

Exemplo


                <p>Minha cor favorita é <del>azul</del> vermelha.</p>

Elemento HTML <ins>

O elemento HTML <ins> define um texto que foi inserido em um documento. Os navegadores geralmente sublinham a inserção do texto

Exemplo


                <p>Minha cor favorita é <del>azul</del> <ins>vermelha</ins>.</p>

Elemento HTML <sub>

O elemento HTML <sub> define um texto subscrito. O texto subíndice aparece meio caractere abaixo da linha normal, e às veses é renderizada em fonte menor. Texto subscrito pode ser usado para fórmulas químics, como H2O

Exemplo:


                <p>Formula quimica da agua H<sub>2</sub>O.</p>

Elemento HTML <sup>

O elemento HTML <sup> define o texto em excepício. O texto em sobrescrito aparece meio caractere acima da linha normal, e às vezes é renderizada em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW1

Exemplo


                <p>Isto é um paragrafo com texto <sup>sobrescrito</sup></p>

Links HTML

Links são definidos com a tag <a>

Exemplo:


                <a href="https://www.w3schools.com">Este é um link.</a>

O destino do link é especificado no atributo href

Atributos são usados para fornecer informações adicionais sobre elementos HTML.

Imagens HTML

Imagens HTML são definidas com a tag <img>.

O arquivo fonte src(), texto alternativo alt(), width e height são fornecidos como atributos.

Exemplo


                <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
