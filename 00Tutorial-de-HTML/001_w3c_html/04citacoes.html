Elementos de citação em HTML

Neste capítulo, vamos passar pelos elementos HTML <blockquote>, <q>, <abbr>, <address>, <cite> e <bdo>

HTML <blockquote> para citações

O elemento HTML <blockquote> define uma seção que é citado de outra fonte.

Os navegadores geralmente recuam elementos <blockquote>

Here is a quote from WWF's website:

Exemplo:


            <p>Aqui está uma citação do site da WWF:</p>
            <blockquote cite="http://www.worldwildlife.org/who/index.html">
            For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation organization, WWF works in nearly 100 countries. At every level, we collaborate with people around the world to develop and deliver innovative solutions that protect communities, wildlife, and the places in which they live.
            </blockquote>

visual:

For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation organization, WWF works in nearly 100 countries. At every level, we collaborate with people around the world to develop and deliver innovative solutions that protect communities, wildlife, and the places in which they live.

HTML <q> para citações curtas

A tag HTML <q> define uma citação curta.

Os navegadores normalmente inserem aspas ao redor da aspas (<q>).

Exemplo:


            <p>WWF's goal is to: <q>Build a future where people live in harmony with nature</q>.</p>

HTML <abbr> para abreviações

A tag HTML <abbr> define uma abreviação ou sigla, como HTML, CSS, JS e PHP

Marcação de abreviações pode fornecer informações úteis para navegadores, tradução sistemas e mecanismos de busca.

Use o atributo global title para mostrar a descrição para a sigla quando você passa o mouse sobre o elemento.

Exemplo


            <p>O <abbr title="World Health Organization">WHO</abbr> foi fundado em 1948.<p>

HTML <address> para informações de contato

A tag HTML <address> define as informações de contato do autor/proprietario de um documento ou artigo

As informações de contato podem ser um endereço de e-mail, URL, endereço fisico, telefone, redes sociais, etc.

O texto no elemento <address> geralmente é renderizado em itálico, e os navegadores também adiciona uma quebra de linha antes e depois do elemento <address>.

Exemplo


            <address>
            Written by John Doe.<br>
            Visit us at:<br>
            Example.com<br>
            Box 564, Disneyland<br>
            USA
            </address>

Visual:

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

HTML <cite> para o titulo da obra

A tag HTML <cite> define o título de uma obra criativa (por exemplo, um livro, um poema, uma musica, um filme, uma pintura, uma escultura, etc.).

NOTA: O nome de uma pessoa não é o titulo da obra.

O texto no elemento <cite> geralmente é renderizado em italico.

Exemplo:


            <p><cite>The Scream</cite> by Edvard Munch. Painted in 1893.<p>

HTML <bdo> para Bi-Directional Override

BOD significa Bi-Direcional Override.

A tag HTML <bdo> é usada para sobrescrever a direção atual do texto:

Exemplo:


            <bdo dir="rtl">This texto will be written from right to left</bdo>

Comentários em HTML

Comentários em HTML não são exibidos no navegador, mas podem ajudar a documentar seu código fonte HTML.

Tag de comentário HTML

Voce pode adicionar comentarios a sua fonte usando a seguinte sintaxe:

Exemplo:


            <!--Escreva seu comentario aqui-->

Note que há um ponto de exclamação (!) na tag inicial, mas não no fim da etiqueta

Com os comentários, voce pode colocar notificações e lembretes no seu código HTML
